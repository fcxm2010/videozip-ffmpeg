# 视频批量压缩工具 (macOS)

这是一个专为 macOS 设计的视频批量压缩 Python 脚本。它利用 FFmpeg 强大的处理能力，结合 AppleScript 提供的原生交互对话框，通过简单的可视化操作即可实现高效的视频瘦身。

## ✨ 主要功能

- **可视化操作**：通过 macOS 原生对话框选择文件夹、编码格式和压缩质量，无需手动输入路径。
- **智能批量处理**：递归遍历指定文件夹下的所有子目录，保持目录结构。
- **多种编码支持**：支持 H.264 (兼容性好) 和 H.265 (压缩率高) 两种主流编码。
- **防膨胀机制**：如果压缩后的文件比原文件还大，脚本会自动保留原文件，确保「只变小，不损画质且不浪费空间」。
- **处理非视频文件**：自动将其他格式的文件（如图片、文档）原样复制到目标目录，实现一键迁移。
- **进度显示**：实时显示当前视频的转码进度百分比。
- **兼容性优化**：
    - 针对 macOS 预览（QuickLook）优化，采用 `yuv420p` 像素格式。
    - 开启 `faststart` 标志，使视频在网络播放时能更快加载。
- **自动断点续传**：如果任务中途停止，下次运行时会自动跳过已处理的文件。
- **详细报告**：任务结束后会生成完整的统计报告，包括处理文件数、节省空间和压缩率。

## 🛠️ 环境要求

1.  **Python 3**：macOS 通常预装了 Python 3。
2.  **FFmpeg**：必须在系统中安装。
    - 推荐使用 [Homebrew](https://brew.sh/) 安装：
      ```bash
      brew install ffmpeg
      ```

## 🚀 使用方法

1.  **克隆或下载脚本**到本地。
2.  **运行脚本**：
    在终端中切换到脚本所在目录，执行：
    ```bash
    python3 compress_video.py
    ```
3.  **交互式设置**：
    - **选择文件夹**：运行后会弹出窗口，请选择需要压缩的视频所在的源文件夹。
    - **选择编码**：选择 `H.265`（推荐，文件更小）或 `H.264`（兼容性极佳）。
    - **设置画质**：输入 CRF 值（0-51）。建议设置为 **23**。
        - 数值越小画质越好，体积越大。
        - 数值越大画质越差，体积越小。
4.  **查看结果**：
    - 脚本会在源文件夹同级目录下创建一个名为 `[原文件夹名]_compressed` 的新文件夹。
    - 所有的压缩视频（统一封装为 `.mp4`）和原始非视频文件都会出现在这里。

## 📝 配置选项

脚本开头的 `配置区域` 可以根据需要灵活修改：

- `VIDEO_EXTENSIONS`: 支持的视频后缀列表。
- `MAX_AUDIO_KBPS`: 音频码率上限，防止音频占用过多空间。

## ⚠️ 注意事项

- **显卡加速**：当前脚本主要使用 CPU (Software) 编码以保证最佳的压缩率和画质。如果需要硬件加速（如苹果 M 芯片的 VideoToolbox），可以修改代码中的编码器参数。
- **文件覆盖**：脚本会自动跳过目标文件夹中已存在的同名文件，请确保目标目录是空的或者您确实想跳过之前的结果。
